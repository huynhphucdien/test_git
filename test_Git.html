How to use Git on Github

Welcome to Github

B1: Tao Repository tren Github

B2: git init: 'khởi tạo git ở thư mục làm việc'

B3: git remote add origin 'URL github' 'Liên kết repo local to repo remote'

B4: git add "file" 'thêm thay đổi của 1 file nhất định vào repo local'
git add . 'add tất cả thay đổi'

B5: git commit -m "decribe message" 'Lưu lại những thay đổi vào workspace'

B6: git commit --ammend -m "New message" 'Thay đổi nội dung của commit'

B7: git push origin [branch-name] 'Upload tất cả commit trong local branch lên github'

git diff b325c da985 'Xem sự khác biệt giữa 2 commit'

git log 'Dùng để list ra những commit mà mình đã thực hiện trên branch'

git reset file 'remove file đó ra khỏi stage area'

git clone [url on github] [create folder] 'Copy 1 dự án trên github về máy'

git pull 'Để đồng bộ với remote repo'

git branch [tên branch] 'Tạo branch mới'

git checkout [branch] 'Chuyển sang làm việc trên branch đó'

git status : xem thong tin
